三更花凉梦醒，缘是泪洒痴情

三更花凉梦醒，缘是泪洒痴情

  • 前世，共赏烟花漫天，痴情红颜；今生，空守三生石畔，奈何桥边。
  • 前世，春榭游丝飘飞，花开为谁；今生，已是烟停水榭，满眼凄美。
  • 前世，你我踱步画廊，雨打西窗；今生，渐觉粉面渺茫，满地花殇。
  • 前世，柔词满袖挥洒，天女散花；今生，空叹一纸韶华，碎成残沙。
  • 前世，醉卧琼林美宴，酒尽言欢；今生，登高独凭画栏，酒尽无言。
  • 前世，只为倾国倾城，一国之兵；今生，尽是朱颜凋零，醉梦不醒。
  • 前世，说好抚琴弄萧，茶凉棋后；今生，偏是竹笛横吹，吹断更漏。
  • 前世，伊人呵手成书，自叹不如；今生，谁再信手挥出，满纸流苏。

倦倚西风夜已昏

倦倚西风夜已昏

  • 莫名的殇。
  • 酒浓惊破残梦里，醉卧衾枕听风雨。
  • 这不是江南氤氲该有的雨，来的那么急，没有一丝顾虑，放肆的冲刷着 一切，却冲不走心里的雾霾。
  • 偷来梨蕊三分白，借的梅花一缕魂，白海棠依旧如此，而吟诗的人却早已不在。 隔世的娇羞默默，也许早已无人在意，却依然痛击我心里。
  • 秋花惨淡秋草黄，耿耿秋灯秋夜长。已觉秋窗秋不尽，那堪风雨助凄凉！助秋风雨来何速！惊破秋窗秋梦绿。
    抱得秋情不忍眠，自向秋屏移泪烛。泪烛摇摇爇短檠，牵愁照恨动离情。谁家秋院无风入？何处秋窗无雨声？
    罗衾不奈秋风力，残漏声催秋雨急。连宵脉脉复飕飕，灯前似伴离人泣。寒烟小院转萧条，疏竹虚窗时滴沥。
    不知风雨几时休，已教泪洒窗纱湿。
  • 伊人听雨的意境，也许无法企及，可这江南水乡的氤氲，又为何为之蒙上一层诉说忧伤的朦胧。缘非如此。
  • 雨点掉落下来，砸在小桥流水的诗里。碎落的枯叶飞舞着砸向浮萍，挑起的波纹恰似揉碎的春心萌动。青春的路走的太漫长，留下的却是满腹的离殇。一句话说的好，为什么我们在青春里走着以后的路，而到了后来，青春走了，我们本该继续走以后的路，却泪雨涟涟的怀念着青春。人生就是如此，路只有一次，怎么走，是否走好，上天不会给你第二次机会，错过就是错过，那就是一生永远的痛，一辈子都弥补不了。
  • 残阳红煞，最宜赌书听风嗅茶香。乱不了的心境，才华依旧，我心依旧。

  • 梦里相寻，却寻回了梦里。原来冰冷的现实不曾拥有。
  • 问候三生石畔，相遇奈何桥边。铮铮琴弦，原是眷恋缠绵，却终是跌碎思念。无聊的寂寞翻开了残句诗篇，苦笑湖边当年，玉指轻拈，轻腕流转，碎了满纸江山。如今一晌贪欢，换了人间，负了似水流年。
  • 似有厢房，藏了谁的流殇？伊人轩窗，紧闭了谁的胸膛？蕊寒清香，穿了谁的柔肠？嫩寒锁梦因春凉，芳气袭人是酒香。寒冷的月光，迷离了谁的泪光？不经意的笔墨，指尖却雕琢出你的面庞。
  • 败墨残笔，还想秀出你的真迹？浪荡才子，谁猜的出你的本意？缘是那缠绵的记忆，堆叠在受伤的心底。
  • 雁阵惊寒，哽咽了华羽清涟。染指江湖，违背了当初誓言。说好的青笺澹水，握卷青花别咫尺，到头来残灯孤梦，漫卷诗书了痴狂。。。

欢迎使用马克飞象

@(示例笔记本)[马克飞象|帮助|Markdown]

马克飞象是一款专为印象笔记（Evernote）打造的Markdown编辑器，通过精心的设计与技术实现，配合印象笔记强大的存储和同步功能，带来前所未有的书写体验。特点概述：

  • 功能丰富 ：支持高亮代码块、LaTeX 公式、流程图，本地图片以及附件上传，甚至截图粘贴，工作学习好帮手；
  • 得心应手 ：简洁高效的编辑器，提供桌面客户端以及离线Chrome App，支持移动端 Web；
  • 深度整合 ：支持选择笔记本和添加标签，支持从印象笔记跳转编辑，轻松管理。

Markdown简介

Markdown 是一种轻量级标记语言，它允许人们使用易读易写的纯文本格式编写文档，然后转换成格式丰富的HTML页面。 —— 维基百科

正如您在阅读的这份文档，它使用简单的符号标识不同的标题，将某些文字标记为粗体或者斜体，创建一个链接或一个脚注[^demo]。下面列举了几个高级功能，更多语法请按Ctrl + /查看帮助。

复选框

使用 - [ ] 和 - [x] 语法可以创建复选框，实现 todo-list 等功能。例如：

  • [x] 已完成事项
  • [ ] 待办事项1
  • [ ] 待办事项2

注意：目前支持尚不完全，在印象笔记中勾选复选框是无效、不能同步的，所以必须在马克飞象中修改 Markdown 原文才可生效。下个版本将会全面支持。

印象笔记相关

笔记本和标签

马克飞象增加了@(笔记本)[标签A|标签B]语法, 以选择笔记本和添加标签。 绑定账号后， 输入(自动会出现笔记本列表，请从中选择。

笔记标题

马克飞象会自动使用文档内出现的第一个标题作为笔记标题。例如本文，就是第一行的 欢迎使用马克飞象。

快捷编辑

保存在印象笔记中的笔记，右上角会有一个红色的编辑按钮，点击后会回到马克飞象中打开并编辑该笔记。

注意：目前用户在印象笔记中单方面做的任何修改，马克飞象是无法自动感知和更新的。所以请务必回到马克飞象编辑。

数据同步

马克飞象通过将Markdown原文以隐藏内容保存在笔记中的精妙设计，实现了对Markdown的存储和再次编辑。既解决了其他产品只是单向导出HTML的单薄，又规避了服务端存储Markdown带来的隐私安全问题。这样，服务端仅作为对印象笔记 API调用和数据转换之用。

隐私声明：用户所有的笔记数据，均保存在印象笔记中。马克飞象不存储用户的任何笔记数据。

离线存储

马克飞象使用浏览器离线存储将内容实时保存在本地，不必担心网络断掉或浏览器崩溃。为了节省空间和避免冲突，已同步至印象笔记并且不再修改的笔记将删除部分本地缓存，不过依然可以随时通过文档管理打开。

注意：虽然浏览器存储大部分时候都比较可靠，但印象笔记作为专业云存储，更值得信赖。以防万一，请务必经常及时同步到印象笔记。
